## 🚀 Setup Instructions

1. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```
2. **Start the development server:**
   ```bash
   npm start
   # or
   yarn start
   ```
3. **Build for production:**
   ```bash
   npm run build
   # or
   yarn build
   ```
